# Fitness-Tracker
A personal fitness tracker application that allows the user to record their workout sessions. Features a Mongo database w/ a mongoose schema and handle routes with Express.


# Technologies Utilized

* HTML
* CSS
* Javascript
* Express
* MongoDB
* Mongoose
* NPM modules
